# Putting the app on your desktop 🖥️

Download the right file for your computer from the
[**Releases page**](https://github.com/Netspecs/slicer-profile-converter/releases),
then follow the steps for your operating system.

---

## 🪟 Windows

**Recommended — use the installer (adds a desktop icon + uninstaller):**
1. Download **`SlicerProfileConverter-Setup.exe`**.
2. Run it and follow the wizard (keep "Create a desktop icon" ticked).
3. You now get a desktop icon, a Start Menu entry, and an uninstaller in
   **Settings → Apps → Installed apps** ("Programs and Features").

> No admin rights needed — it installs just for your user.

**Alternative — portable exe (no install):**
Download `SlicerProfileConverter-windows.exe` and run it directly. To make a
desktop shortcut, put `windows-create-desktop-shortcut.bat` in the same folder
and double-click it.

> 💡 First run may show a SmartScreen prompt → **More info → Run anyway**
> (normal for apps that aren't code-signed).

---

## 🍎 macOS

1. Download `SlicerProfileConverter-macos`.
2. Move it to **Applications** and drag it to your Dock.

> First open: right-click → **Open** → **Open** to bypass the "unidentified
> developer" warning (only once).

---

## 🐧 Linux

1. Put `linux-install.sh` in the same folder as `SlicerProfileConverter-linux`.
2. Run: `bash linux-install.sh`
3. The app appears in your applications menu and on your desktop.
